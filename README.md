## Getting Started

1. Install dependencies:
   ```bash
pnpm install
   ```

2. Start the development server:
```bash
pnpm start
```

3. Run on Android:
   ```bash
pnpm android
```

4. Run on iOS:
```bash
pnpm ios
```
